API Documentatie

Hieronder staat een overzicht van de beschikbare API-endpoints:

Users

  • GET /users: Haal alle gebruikers op.
  • GET /users/:id: Haal een specifieke gebruiker op op basis van ID.
  • POST /users: Voeg een nieuwe gebruiker toe. Vereist: name, email, password.
  • PUT /users/:id: Update een bestaande gebruiker. Vereist: name, email, password.
  • DELETE /users/:id: Verwijder een gebruiker op basis van ID.

Newsposts

  • GET /newsposts: Haal alle nieuwsberichten op. Optioneel: queryparameters title, author, limit, offset.
  • GET /newsposts/:id: Haal een specifiek nieuwsbericht op op basis van ID.
  • POST /newsposts: Voeg een nieuw nieuwsbericht toe. Vereist: title, content, author.
  • PUT /newsposts/:id: Update een bestaand nieuwsbericht. Vereist: title, content, author.
  • DELETE /newsposts/:id: Verwijder een nieuwsbericht op basis van ID.
